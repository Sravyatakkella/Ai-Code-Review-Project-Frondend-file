# AI Code Review 🚀
A futuristic AI-powered code review web application that analyzes source code for syntax errors, security vulnerabilities, performance issues, code quality problems, and spelling mistakes across multiple programming languages.
## 🌐 Live Demo
https://jolly-kheer-6df079.netlify.app/
# ✨ Features
* 🔍 Instant code analysis
* 🛡 Security vulnerability detection
* ⚡ Performance issue detection
* ✨ Code quality analysis
* 🧠 Syntax and spelling checks
* 🌐 Supports 20+ programming languages
* 🎨 Modern futuristic UI
* 📱 Responsive design
* 🚀 Fully frontend-based
* 💻 No backend or API required

# 🔎 Analysis Capabilities

The application can detect:

## Security Issues

* Unsafe `eval()` usage
* Dangerous `innerHTML`
* Hardcoded passwords
* Risky runtime execution

## Performance Issues

* Infinite loops
* Excessive console logs

## Code Quality Issues

* Syntax errors
* Missing semicolons
* Unbalanced brackets
* Invalid structures
* Misspelled keywords
* Undefined variables
* Missing HTML attributes
* Invalid CSS blocks
